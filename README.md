# Cruise Ships :ship:
### Javascript Object-Oriented code for a Cruise Ships.
### Building and working with many objects that allow a cruise ship to operate.

### Running the app
To run clone to local machine.

Navigate to directory and install npm:
```
npm install
```
To install jest:
```
npm install -D jest
```
To run tests:
```
npm run test
```
### Built Using
 - Node.js
 - Jest
